# Daily Dev Log

## 느낀 점 & 회고 (Thoughts & Reflections)

#### 1. Optional
- `Optional` 은 값을 감싸기 위한 Wrapper(래퍼) 객체다
- 값이 null 일수도 있음을 명시적으로 표현한다

```java
Optional<Question> oq = questionRepository.findById(1);
oq.ifPresent(testQ -> System.out.println(testQ.getSubject()));
```

- **깨달은 점 (Realization):**
- 값이 없을 경우: `Optional.empty()` (null 값을 담는 것이 X)
