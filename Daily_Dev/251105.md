# Daily Dev Log

## 느낀 점 & 회고 (Thoughts & Reflections)

#### 1. 다양한 관점
```java
int left = 0;
int right = arr.length -1;

while(left < right) {
    int temp = arr[left];
    arr[left] = arr[right];
    arr[right] = temp;

    left++;
    right--;
}
```

- **깨달은 점 (Realization)**
  index 양 끝에서 하나씩 증감을 하면 홀수 포함 깔끔하게 reverse가 가능했다. 나는 새로운 배열을 만들어서 역순으로 데이터를 넣었다. 이 방법도 함께 알고 있으면 좋을 것 같아서 기록한다.


#### 2. Base의 중요성


`public 반환타입` , 변수명 앞에는 클래스명이 들어가는 것 또한 변수 선언을 생각하면 당연하다.
